Exceptions: TRY, CATCH, FINALLY, and RAISE

Basil supports structured exception handling. See the full guide: docs/guides/EXCEPTIONS.md

Quick example:

TRY
  IF x% = 0 THEN RAISE "Divide by zero"
  PRINT 10 / x%
CATCH err$
  PRINT "Oops: ", err$
FINALLY
  PRINT "Always runs"
END TRY

Notes:

  • CATCH variable (if present) must be a string (use '$' suffix).
  • FINALLY always runs on normal or exceptional exit.
  • Use RAISE with no expression inside CATCH to rethrow.
